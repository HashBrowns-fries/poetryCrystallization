## 一、问题诊断（审计阶段）

### 1.1 原代码（`_archive/40_genre_dominance.py`）的两层错误

**错误 1：置换检验逻辑错配**（第 135-147 行）

```python
def perm_test(cond_ss_func, n_p=N_PERM):
    cnt = 0
    for _ in range(n_p):
        perm_fB = fB_lbl.copy()
        np.random.shuffle(perm_fB)
        jp = np.array([f"{a}_{b}" for a, b in zip(fA_lbl, perm_fB)])
        ss_jp  = ss_between(jp)
        ss_Bp  = ss_between(perm_fB)
        cp     = max(0, ss_jp - ss_b_A - ss_Bp)   # ← 这是【交互效应】SS
        if cp >= cond_ss_func: cnt += 1            # ← 比较【纯效应】观测值
    return round(cnt / n_p, 4)

p_A = perm_test(ss_A_cond)   # 检验纯体裁效应？
p_B = perm_test(ss_B_cond)   # 检验纯朝代效应？
```

函数固定 fA、置换 fB，算的是 **交互效应** 的置换 SS，但用来比较 **纯效应** 观测值。两个量量级差异巨大（180,415 vs ≈0），导致 p 值脱离统计意义。

**错误 2：SS 计算框架有偏差**

旧代码用朴素 PERMANOVA 公式：
```
SS_between(labels) = SS_Total - Σ_groups SS_within
```

这个公式在 **单因素** 下与 PCoA 结果等价，但在 **两因素共线** 下会因为 sub-cell 计数膨胀产生虚假 SS。例如观测的 14 个 sub-cell（3×7-7=14，因体裁朝代共线削减）在置换后扩张到 21 个 sub-cell，SS_joint 系统性偏高。

**修正办法**：改用 vegan::adonis2 的标准实现 —— McArdle & Anderson 2001 的 PCoA 投影：

$$SS_X = \text{trace}\left((X^\top X)^{-1} X^\top G X\right) - SS_{\text{grand}}$$

其中 $G = -\frac{1}{2}(I - \frac{1}{n}\mathbf{1}\mathbf{1}^\top) D^2 (I - \frac{1}{n}\mathbf{1}\mathbf{1}^\top)$ 是 Gower 中心化矩阵。

### 1.2 健全性测试

我们在小规模合成数据（n=100）上对比两种实现：

- 朴素 SS + unrestricted permutation：A 是已知强信号，p_A 算得 1.0000（完全反直觉）
- PCoA + unrestricted permutation：A 强信号 p<0.001，纯随机变量 C \| A 给 p≈0.84（uniform）

PCoA 框架是正确的实现。

### 2.3 关键修正

1. **R² 全面下降**：旧版 R² 普遍虚高（朴素 SS 在共线场景下偏大）。新版 R² 在国际计算诗学/生态学文献的常见量级范围内。

2. **朝代条件效应：从"不显著"修正为"极显著"**：原报告 p=0.283 是 bug 产物。

3. **体裁条件效应：仍显著，但量级修正**：从 R²=0.014 上升到 R²=0.075；F=246.80。

4. **朝代 / 体裁的相对量级**：旧版 R²=0.733 vs 0.014 给出虚假的"5.2×碾压"印象；新版 0.203 vs 0.075 是 2.7×，朝代仍主导但不至于压倒。

## 六、教训

1. **审稿人的统计质疑要逐字核对**，不能用 hand-wave 的"散布差异/F 效率不足"敷衍。R²=0.733 与 p=0.283 是数学上的硬冲突，必须用代码层面的证据回应。

2. **朴素 PERMANOVA 公式（SS_T - Σ within）在两因素共线场景下不可靠**。这是社区常识但容易被忽略。McArdle-Anderson PCoA 框架是 vegan::adonis2 实际实现，是提交分析报告的应有标准。

3. **"看起来 p 值奇怪"应当立即触发审计**，不应当假设结果只是"反直觉但合理"。今日的 bug 在原版于 2025 年完成时就存在，半年来一直被错误地用 PERMDISP 解释合理化。

4. **新结果其实更稳**。修正后朝代 / 体裁条件 R² 比从 5.2× 降到 2.7×，并未颠覆论文的方向，反而让叙事更可信。
